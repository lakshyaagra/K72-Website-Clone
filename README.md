# K72 Website Clone

🚀 **Live Demo:** [View Website](https://k72-website-iwdx.onrender.com/)

## About

This project is a pixel-perfect clone of the K72 website built using React.js,TailwindCSS and GSAP Animations. The goal was to recreate the original design, layout, responsiveness, and user experience while following modern frontend development practices.

## Features

- Fully responsive design
- Modern UI built with Tailwind CSS
- Handled Animations using GSAP Animations
- Component-based architecture using React.js
- Smooth layout and styling recreation
- Mobile, tablet, and desktop support

## Tech Stack

- React.js
- Tailwind CSS
- GSAP Animations
- JavaScript
- Vite
